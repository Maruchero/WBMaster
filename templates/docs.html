# Django

Cosa è

Django è un framework web di alto livello scritto in Python che incoraggia lo sviluppo rapido e il design pulito e pragmatico. È stato creato da sviluppatori esperti e si occupa di gran parte delle difficoltà dello sviluppo web, così puoi concentrarti sulla scrittura della tua app senza dover reinventare la ruota.

Django è stato creato nel 2003 da Adrian Holovaty e Simon Willison come framework interno per il giornale online Lawrence Journal-World. Nel 2005 è stato rilasciato come open source sotto licenza BSD. Il nome Django è un omaggio al chitarrista jazz Django Reinhardt

Caratteristiche

Django ha diverse caratteristiche:

  • Django ha l'architettura Model-View-Template (MVT) che separa la logica di business, la presentazione e l'accesso ai dati. Questo rende il codice più pulito, modulare e riutilizzabile.
  • Django è altamente sicuro e aiuta gli sviluppatori a evitare molti errori comuni come il clickjacking oppure il cross-site request forgery ma anche l'SQL injection. Inoltre, fornisce un sistema di autenticazione che gestisce in modo sicuro gli account e le password degli utenti.
  • È alimentato da Python, un inguaggio compilato
  • Fornisce un pannello di amministrazione predefinito che facilita la gestione del sito web.
  • È buono per l’ottimizzazione dei motori di ricerca (SEO), in quanto consente di creare URL user-friendly e personalizzabili.
  • Fornisce un Object Relational Mapper (ORM) che semplifica il lavoro con il database e i dati.

Svantaggi

Django ha anche diversi svantaggi:

  • Django richiede di scrivere le espressioni regolari per definire gli URL delle applicazioni, il che può essere difficile per i principianti o per chi non è abituato a questo tipo di sintassi.
  • Django è un framework molto opinionato, che impone una certa struttura e un certo modo di fare le cose. Questo può essere un vantaggio per la coerenza e la standardizzazione, ma anche uno svantaggio per la flessibilità e la personalizzazione.
  • Django non supporta nativamente la gestione di più richieste contemporaneamente, perché usa un modello sincrono basato su thread. Questo può limitare le prestazioni e la scalabilità delle applicazioni che devono gestire un alto traffico o operazioni asincrone.

Installazione

Per installare Django, devi avere Python installato sulla tua macchina. Puoi verificare la versione di Python eseguendo py --version nel prompt dei comandi. Poi devi creare un ambiente virtuale per il tuo progetto usando il comando py -m venv project-name. Per attivare l'ambiente virtuale, esegui project-name\Scripts\activate.bat. Infine, puoi installare Django usando pip, il gestore di pacchetti di Python, con il comando py -m pip install Django. Questo scaricherà e installerà l'ultima versione di Django. Puoi verificare la tua installazione di Django eseguendo django-admin --version nel prompt dei comandi.

Per aggiornare la versione di django puoi eseguire python -m pip install -u Django nel prompt dei comandi
